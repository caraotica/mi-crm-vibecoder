# Clientes

Componentes de las pantallas de Lista de clientes y Ficha de cliente.

- `ClienteListItem.tsx` — fila con avatar, nombre, último contacto y chip de
  estado comercial (WUA-9).
- `ClienteFichaHeader.tsx` — avatar, nombre, empresa, chips de estado/origen,
  teléfono/email tappables (WUA-11, WUA-63).
- `AccionesRapidas.tsx` — 3 botones-tarjeta (Anotar interacción / Programar
  seguimiento / Registrar venta) — fila en escritorio, columna en móvil (WUA-11).
- `HistorialTimeline.tsx` — combina interacciones, ventas y seguimientos
  completados en orden cronológico inverso (WUA-11, WUA-16).
