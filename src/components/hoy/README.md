# Hoy

Componentes de la pantalla de inicio.

- `QuickActionsPanel.tsx` — rejilla de 4 tiles (Nueva tarea, Anotar
  interacción, Registrar venta, Nuevo cliente) — 4 columnas escritorio / 2×2
  móvil (WUA-17).
- `SeguimientoSection.tsx` — cabecera de sección (punto de color + título +
  contador) para Atrasados / Para hoy / Próximas (WUA-17, WUA-18).
- `SeguimientoItem.tsx` — checkbox "Hecho", nombre + chip de estado del
  cliente, avatar del responsable, fecha relativa (WUA-17, WUA-22).
